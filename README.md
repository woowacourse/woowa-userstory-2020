# 📝 2. 유저 스토리 도출하기
## 1)시나리오
### (1)서비스 사용 전
> 세연은 아들이 낮잠을 자는 동안 내일 할 성만들기 놀이를 생각했다. 재료는 인터넷으로 주문했는데, 오늘 도착할 예정이었다. 성은 아들이 손재주가 있어 3일이면 완성될 것 같았다. 그 이후에 어떤 놀이를 할 지 아이디어가 떠오르지 않아 자주 들어가던 맘카페에 들어갔지만 추천글과 댓글에는 예전에 해 본 것들 뿐이었다. 아들이 해봤던 것은 금방 질려하는 탓에 새로운 놀이를 찾아야만 했다. 포털에서 '엄마표 놀이'로 검색하자 다른 카페들의 글들이 나왔고, 꽤 흥미로운 제목의 글이 있었다. 하지만 글을 읽으려면 카페에 새로 가입을 해야했다. 맘카페들은 인증과정이 꽤 복잡해 바로 가입할 수도 없었다. 세연은 한숨을 쉬며, 쇼핑어플을 켰다. 장난감 카테고리에서 인기순으로 검색해봤지만 1위부터 30위의 상품들 모두 집에 있는 것들이었다. 이번에는 신상품순으로 검색했다. 스크롤을 내리다보니 중간쯤에 눈에 띄는 제품이 있었다. 하지만 그간 구매해온 장난감들에 비해 꽤나 비싼 것 같았다. 조금이라도 아끼고자 하는 마음에, 세연은 다시 포털에서 해당 제품을 검색했다. 몇가지 쇼핑몰들이 나왔고, 각 쇼핑몰마다 들어가보며 가격을 조사하던 중 아들이 잠에서 깼다.

### (2)서비스 사용 후
> 세연은 아들이 낮잠을 자는 동안 내일 할 성만들기 놀이를 생각했다. 재료는 엄마손(서비스 가명)에서 주문했는데, 오늘 오후에 도착한다는 알림이 왔다. 손재주가 있어 3일이면 완성될 것 같았다. 그 이후에 어떤 놀이를 할 지 아이디어가 떠오르지 않아 엄마손에 들어갔다. 이미 해본 놀이를 필터로 제외한 뒤 다른 엄마들의 추천순으로 검색을 한 결과가 나왔다. 그 중 상위에 랭크된 놀이가 마음에 들어 클릭했다. 관련 영상들과 필요한 재료들을 구매할 수 있는 곳들의 가격이 비교되어 있었다. 가격과 포인트적립등을 고려했을 때 가장 합리적인 곳에서 구매를 완료한 뒤 아들이 깰 때까지 유튜브에서 화초 관리법에 대한 영상을 봤다. 며칠 뒤 구매했던 재료가 도착하는 날 관련 영상의 링크가 알림으로 도착해, 바로 들어가 볼 수 있었다.
